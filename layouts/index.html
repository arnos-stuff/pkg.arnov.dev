What this website is about

As a hobbyist, I have a few different goals at the moment.
First, I want to keep learning data-science-y stuff using Python and Julia, from a pure data perspective:
fitting models, seeing AUCs, having a notion of whether that new architecture is really great etc.
Second, I want to get into DevOps and Full-Stack dev a tiny bit more: More JS for the dataviz and the front to really share my passion with others, and rust for the backend because of python bindings and great CLIs

In terms of topics, I'm going to be focusing on a few things that matter most to me at the moment:
First, I want to learn economics, like actual economics. I want to reproduce studies, compute equations to figure out if the models are whack or not.
Second, I want to get into Deep Learning again, especially NLP. I feel like there are LLM applications that are so fun and easy to make it's just lost potential not to try !
Third, I want to get into real dataviz, using javascript, and geospatial data, using TopoJSON and GeoJSON data. I've got a small start already ðŸ˜Ž
Finally, as a more tech topic, I'd like to be able to run cloud/remote workflows well. I've got a GCP subscription, everything ready ðŸš€
